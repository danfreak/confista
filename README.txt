=======
README
=======

Confista_ (Conference + Assistant) is a very simple and extensible
conference management application built on top of the awesome
Pinax_, a platform for rapidly developing websites.

Check the ``docs/INSTALL.txt`` file for details regarding the 
installation.

.. _Confista: http://github.com/theju/confista/tree/master
.. _Pinax: http://pinaxproject.com/
